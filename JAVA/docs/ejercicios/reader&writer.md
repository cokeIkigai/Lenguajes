# Ejercicio Reader & Writer

- Crea varios archivos que contengan:

  - Usuarios
  - Libros disponibles 
  - Libros prestados
 
  ---

Ejemplo:
  
**usuarios.txt**

```
=== BIBLIOTECA MUNICIPAL "NOMBRE DE LA BIBLIOTECA" ===
ARCHIVO: REGISTRO DE USUARIOS
Fecha creación: 15/03/2024
Formato: ID|Nombre|Email|Estado|Fecha_Registro (ejemplo)
```

**libros.txt**

```
=== BIBLIOTECA MUNICIPAL "NOMBRE DE LA BIBLIOTECA" ===
ARCHIVO: CATÁLOGO DE LIBROS
Fecha creación: 15/03/2024
Formato: ISBN|Título|Autor|Género|Año|Estado|ID_Usuario|Ubicación (ejemplo)
====================================================================

```

**prestamos.txt**

```
=== BIBLIOTECA MUNICIPAL "MIGUEL DE CERVANTES" ===
ARCHIVO: REGISTRO DE PRÉSTAMOS
Fecha creación: 15/03/2024
Formato: ID_PRESTAMO|ISBN|ID_Usuario|Fecha_Inicio|Fecha_Devolución|Estado|Bibliotecario  (ejemplo)
===============================================================================

```
