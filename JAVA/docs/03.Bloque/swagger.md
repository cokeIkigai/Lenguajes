# ⭐ Qué es Swagger

Swagger es una `herramienta visual` que permite ver y probar tu API desde una página web.

- No es un backend.
- No es una API.
- Es un front-end generado automáticamente.

#### ⭐  ¿Qué hace Swagger exactamente?
1. `Muestra` tu API en una interfaz web ✔️

2. Swagger `analiza` tus controllers de Spring Boot y genera una página como esta:✔️

3. Lista de `endpoints`✔️

4. Tipos de `peticiones` (GET, POST, PUT, PATCH, DELETE)✔️

   - Parámetros
   - Cuerpos JSON
   - Respuestas
   - Códigos de estado
   - Todo organizado de forma visual.

5. Genera documentación automática ✔️

6. Swagger crea documentación que describe tu API para:✔️
   - Desarrolladores
   - Testers
   - Clientes
   - Equipos de frontend

*La documentación siempre está actualizada porque se genera desde tu código.* ✔️

*No es tu front-end real. Es una herramienta accesoria para desarrollo.*✔️
