### TOP 10 ESENCIALES

|**Atajo**|**Descripción**|
|---------|---------------|
|Ctrl + S	                        | Guardar (siempre funciona)|
|Ctrl + Z / Ctrl + Y	            |Deshacer / Rehacer|
|Ctrl + C / Ctrl + V / Ctrl + X	  | Copiar / Pegar / Cortar|
|Ctrl + F	                        | Buscar en archivo|
|Alt + Enter	                    | Quick fixes (correcciones rápidas)|
|Ctrl + Space	                    | Autocompletar código|
|Ctrl + B o Ctrl + Click	        | Ir a definición |
|Ctrl + Shift + F10	              | Ejecutar clase actual |
|Ctrl + /	                        | Comentar/descomentar línea |
|Ctrl + D	                        | Duplicar línea |

### CODIFICACIÓN RÁPIDA

|**Atajo**|**Descripción**|
|---------|---------------|
|Ctrl + Alt + L	                  |Formatear código (auto-indentar)|
|Ctrl + Shift + ↑ / ↓	            |Mover línea/bloque|
  |Ctrl + Y	                      |Eliminar línea|
|Shift + Enter	                  |Nueva línea (sin mover cursor)|
|Ctrl + W / Ctrl + Shift + W	    |Expandir/reducir selección|
|Ctrl + Shift + J	                |Unir líneas|
|Ctrl + Alt + O	                  |Organizar imports|

### NAVEGACIÓN Y BÚSQUEDA

|**Atajo**|**Descripción**|
|---------|---------------|
|Ctrl + N	                    | Buscar clase|
|Ctrl + Shift + N	            | Buscar archivo|
|Ctrl + E	                    |Archivos recientes|
|Ctrl + B	                    |Ir a declaración|
|Ctrl + Alt + B	              |Ir a implementación|
|Ctrl + F12	                  | Ver estructura del archivo|
|Ctrl + Shift + A	            | Buscar acción/comando|

### REFACTORIZACIÓN

|**Atajo**|**Descripción**|
|---------|---------------|
|Shift + F6	            | Renombrar (variable/clase/método)|
|Ctrl + Alt + V	        |Extraer variable|
|Ctrl + Alt + M	        |Extraer método|
|Ctrl + Alt + P	        |Extraer parámetro|
|Alt + Insert	          |Generar código (getters, constructores)|

###  DEBUGGING

|**Atajo**|**Descripción**|
|---------|---------------|
|Shift + F9	        |Iniciar debug|
|F8	Step Over       |(avanzar)|
|F7	Step Into       |(entrar en método)|
|Shift + F8	        |Step Out (salir de método)|
|Alt + F8	          |Evaluar expresión|
|Ctrl + F2	        |Detener ejecución|

### VENTANAS Y HERRAMIENTAS

|**Atajo**|**Descripción**|
|---------|---------------|
|Alt + 1	          |Mostrar/ocultar panel Project|
|Alt + F12	        |Abrir terminal|
|Ctrl + Tab	        |Cambiar entre pestañas|
|Ctrl + F4	        |Cerrar pestaña|
|Ctrl + Shift + F12	|Maximizar editor|
|Esc	              | Volver al editor|

### Acciones importantes

|**Atajo**|**Descripción**|**Categoría**|
|---------|---------------|-------------|
|Ctrl + Shift + F	        |Buscar en TODO el proyecto	        |Búsqueda|
|Ctrl + Shift + R	        |Reemplazar en TODO el proyecto	    |Búsqueda|
|Ctrl + Shift + F10	      |Ejecutar clase actual	            |Ejecución|
|Ctrl + Shift + Enter	    |Completar estructura (; })	        |Código|
|Ctrl + Shift + Space	    |Autocompletar inteligente	        |Código|
|Ctrl + Shift + A	        |Buscar cualquier acción	          |Navegación|
|Ctrl + Shift + Alt + J	  |Seleccionar TODAS ocurrencias	    |Selección|
|Ctrl + Shift + Backspace	|Ir a último cambio	                |Navegación|

### Herramientas avanzadas

|**Atajo**|**Descripción**|**Categoría**|
|---------|---------------|-------------|
|Ctrl + Alt + L	    |Formatear código	              |Formato|
|Ctrl + Alt + O	    |Optimizar imports	            |Código|
|Ctrl + Alt + S	    |Abrir Settings	                |Config|
|Ctrl + Alt + T	    |Rodear código con (if/try)	    |Código|
|Ctrl + Alt + V/M/F	|Extraer variable/método/campo	|Refactor|
